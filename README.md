During my internship, I participated in developing an **Order Management System** using Java Spring Boot and PostgreSQL.

#### Main Responsibilities

* Developed and maintained RESTful APIs using Spring Boot
* Designed database entities and relationships
* Implemented authentication and role-based authorization
* Built order and order-item management features
* Implemented pagination, filtering and keyword searching
* Handled transactional consistency and concurrent inventory updates
* Developed CSV product import and Excel order export
* Created database migrations using Flyway
* Wrote unit tests using JUnit 5 and Mockito
* Supported application deployment and issue investigation

✓ CSV import and Excel export
✓ Database migration with Flyway
✓ Unit testing with JUnit 5 and Mockito
